# AgriVision---Intelligent-Plant-Disease-Detection
🌿 AgriVision – Intelligent Plant Disease Detector

AgriVision is a simple AI-powered application that detects plant diseases from leaf images and provides treatment recommendations.
The application allows users to upload plant leaf images through an interactive GUI and receive disease predictions along with remedies.

---

🚀 Features

- Upload plant leaf images
- Detect plant diseases
- Display prediction confidence
- Suggest remedies for infected plants
- Simple graphical interface

---

🧠 Tech Stack

- Python
- Taipy GUI
- NumPy
- Pillow

⚙️ Installation

Clone the repository:

git clone https://github.com/abhinavX22/agrivision-disease-detector.git
cd agrivision-disease-detector

Install required dependencies:

pip install -r requirements.txt

---

▶️ Run the Application

cd src
python main.py

Then open your browser and go to:

http://127.0.0.1:5000

---

📸 Demo

Upload a plant leaf image and the system will:

1. Display the uploaded image
2. Predict the plant disease
3. Show confidence score
4. Suggest treatment recommendations
